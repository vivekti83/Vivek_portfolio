# 🧬 Vivek Tiwari — Biotechnology Research Portfolio

A modern, responsive biotechnology portfolio website showcasing academic projects, research interests, technical skills, and certifications.

Built with a research-oriented UI inspired by molecular biology, genomics, and life science visualization.

## 📌 About This Project

This portfolio is designed specifically for a Biotechnology student to present:

- Academic & research projects
- Lab techniques and practical exposure
- Bioinformatics skills
- Certifications from trusted platforms
- Career interests in life sciences

The design focuses on a **clean scientific aesthetic** rather than a typical tech portfolio.

---

## 🎯 Key Features

- 📱 Fully Responsive (Mobile, Tablet, Desktop)
- 🧬 DNA + Molecular Theme UI
- 🧪 Research-Oriented Layout
- 🧊 Glassmorphism Cards
- 🌗 Dark/Light Mode Toggle
- 🧾 Academic Project Sections
- 🎓 Certificate Showcase with Verification Buttons
- 🔬 Scientific Visual Elements (Hexagon Grid, Molecule Nodes)
- 🚫 Text Selection Disabled for Better UX
